## php mvc 框架

### 安装
    git clone git@github.com:jyiL/php-mvc.git
    vi config/config.php // 修改数据库配置
    chmod +x install.sh
    ./install.sh    // 创建数据库表
    
### Browse（浏览器）
    http://domain
